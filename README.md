## 💁 프로젝트 소개

'오늘 뭐 입지?'는 날씨에 적합한 옷차림을 고민하는 사람들을 위한 웹 애플리케이션으로, 특히 날씨 변화가 잦은 간절기에 유용하게 쓸 수 있도록 개발하였습니다. 사용자는 오늘의 옷차림, 지역과 함께 간단한 리뷰를 남길 수 있습니다. 그리고 다른 사용자가 공유한 리뷰를 참고해 오늘의 옷차림을 정할 수 있습니다.
